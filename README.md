# Lotus Tours & Travels

Premium luxury travel and vehicle rental website.

## Tech Stack
- React 18 (JavaScript)
- Vite 5
- Tailwind CSS 3
- Framer Motion 11

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push to GitHub
2. Import project on vercel.com
3. Framework: Vite — build command: `npm run build`, output: `dist`
4. Deploy ✓

## Deploy to Netlify

1. Push to GitHub
2. New site from Git on netlify.com
3. Build command: `npm run build`, publish directory: `dist`
4. Deploy ✓

## Content Updates

All content is in `src/data/siteContent.js` — edit copy, fleet, services, testimonials, FAQs, and contact info there.
